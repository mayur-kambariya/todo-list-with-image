# README

This README would normally document whatever steps are necessary to get the
application up and running.

## _Application setup_

1) ```git clone <url>```
2) ```cd todo-list-with-image```
3) ```bundle install```
4) change database.yml with postgres database Credentials
5) ```rails db:create```
6) ```rails db:migrate```
7) ```yarn install```
8) ```rails s```
